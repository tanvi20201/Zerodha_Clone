# Zerodha Clone

A responsive frontend clone of the Zerodha trading platform built using modern web technologies.
This project recreates the core layout and user interface of Zerodha’s website, including landing pages, product information, pricing sections, and support pages.

## 🚀 Project Overview

The objective of this project is to replicate the user experience of the Zerodha platform while practicing component-based UI development and frontend architecture using React.

The application demonstrates structured folder organization, reusable components, and modern frontend practices.

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/tanvi20201/Zerodha_Clone.git
```

Navigate to the project folder:

```
cd Zerodha_Clone/frontend
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm start
```

The application will run at:

```
http://localhost:3000
```
